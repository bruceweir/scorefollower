Class PerformanceMatcher.Onset

class PerformanceMatcher.Onset extends Object

Inner class for representing the corresponding note onset times in two audio files. Each Onset object represents a score position, with a beat number (numbered from 0 at the beginning of the first full bar), and the time (in seconds) of the average onset time of the notes at this score position in the respective audio files.
