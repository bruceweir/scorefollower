public void recalculatePathCostMatrix(int r1, int c1, int r2, int c2)
Calculates a rectangle of the path cost matrix so that the minimum cost path between the bottom left and top right corners can be computed. Caches previous values to avoid calling find() multiple times, and is several times faster as a result.

Parameters: r1 the bottom of the rectangle to be calculated c1 the left side of the rectangle to be calculated r2 the top of the rectangle to be calculated c2 the right side of the rectangle to be calculated
